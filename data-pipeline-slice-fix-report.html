Root Cause: subarray()의 버퍼 공유 문제

핵심: Float32Array.subarray()는 원본 ArrayBuffer의 뷰를 생성합니다. postMessage(msg, [positions.buffer])로 Transferable 전송 시, 뷰가 가리키는 전체 원본 버퍼가 전송됩니다.

Fix: 1-Line Change

// src/utils/rangeImage.ts — convertRangeImageToPointCloud() const maxPoints = height * width const output = new Float32Array(maxPoints * POINT_STRIDE) // ... conversion loop ... − const positions = output.subarray(0, pointCount * POINT_STRIDE) + const positions = output.slice(0, pointCount * POINT_STRIDE) return { positions, pointCount }
