# AV Triage

Encord integration for autonomous vehicle scenario review. Browse scored driving scenarios from Argoverse 2 and Waymo, preview in a 3D LiDAR viewer, and send to Encord for labeling with pre-drawn 3D cuboid predictions.

## Features

- **Scenario browser** — 165+ pre-scored AV scenarios ranked by pedestrian density, cyclist interactions, and near-miss events
- **3D LiDAR viewer** — Point clouds, camera images, bounding boxes, trajectory trails, and keypoint skeletons in the browser
- **Encord integration** — Batch-select scenarios and send to Encord with one click. Waymo scenarios upload as 3D LiDAR scenes with cuboid predictions. AV2 scenarios upload camera frames.
- **Workflow status** — Live status badges (Queued / In Review / Complete) pulled from Encord's workflow API
- **Multi-dataset** — Argoverse 2 (live), Waymo v2 (live), nuScenes (live), with scored scenario index

## Setup

### Prerequisites

- Node.js 20.19+
- Python 3.11
- Encord SSH key at `~/.encord/encord-av-triage-private-key.ed25519`
- GCP credentials for Waymo LiDAR upload (`gcloud auth login`)

### Install

```bash
# Frontend
npm install

# Backend
pip3.11 install fastapi uvicorn boto3 pyarrow encord open3d numpy google-cloud-storage requests
```

### Run

```bash
# Terminal 1 — frontend dev server
npm run dev

# Terminal 2 — API server (Encord bridge)
python3.11 -m uvicorn api.server:app --port 8001 --reload
```

Open http://localhost:5173

### Environment variables (optional)

```bash
ENCORD_SSH_KEY=/path/to/your/key.ed25519    # default: ~/.encord/encord-av-triage-private-key.ed25519
ENCORD_PROJECT_HASH=...                      # default: 1b44da5a-ad5d-425c-818b-014be4dbce14
ENCORD_DATASET_HASH=...                      # default: 25abe913-d0eb-4134-be8d-29712a8354b0
```

## Architecture

```
src/                          # React + Three.js frontend
  components/ScenarioPanel/   # Sidebar with scenario cards, batch send, workflow badges
  components/LidarViewer/     # 3D point cloud viewer
  components/CameraPanel/     # Camera image panels
  components/Timeline/        # Playback controls
  stores/useSceneStore.ts     # Zustand state management
  stores/useFilterStore.ts    # Shared filter state
  data/scenario_index.json    # 165 scored scenarios

api/server.py                 # FastAPI backend
  POST /api/encord/send       # Upload scenario to Encord (image or 3D LiDAR)
  GET  /api/encord/status     # Query workflow statuses
  GET  /api/encord/health     # Health check
```
